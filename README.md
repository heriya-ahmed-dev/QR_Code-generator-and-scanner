This is a simple and responsive QR Code Generator Web App built using HTML, CSS, and JavaScript. It allows users to instantly generate QR codes for any text, URL, or data using a free QR code API.
The app takes user input, converts it into a QR code using an external API, and displays the generated QR code in real time. It is lightweight, fast, and beginner-friendly.

✨ Features
🔹 Generate QR codes instantly from text or URLs
🔹 Uses free QR Code API for generation
🔹 Real-time QR code preview
🔹 Simple and lightweight design
